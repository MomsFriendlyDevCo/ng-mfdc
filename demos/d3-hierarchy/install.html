1. Install the component via bower: 
    bower install angular-venn
  2. Install the pre-req script tags in your application header: 
    <script src="/bower_components/d3/d3.min.js"></script>
<script src="/bower_components/d3.chart/d3.chart.min.js"></script>
<script src="/bower_components/d3.chart.layout.hierarchy/d3.chart.layout.hierarchy.min.js"></script>
<script src="/bower_components/angular-d3-hierarchy/angular-d3-hierarchy.js"></script>
  3. Setup the component in your Angular app 
    angular.module('app', ['angular-d3-hierarchy']);
